# Doctor Managerment
This project is a small part extracted from PRO192 at the university, using JavaFX.

## Environement:
- Linux | Tested | Compiled
- Windows 11 Updating  
- Java 17.0 above

## Usage
1. Install maven
```sh
sudo apt install maven
```
check the maven version
```sh
mvn -v
```
2. Run project
```sh
bash setup.sh
```
